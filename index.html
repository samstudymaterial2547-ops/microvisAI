How it works
01
Upload image
Drag and drop or select a microscope photograph of your bacterial specimen in PNG, JPG, or TIFF format
02
Preprocessing
Pillow resizes the image to 224×224px, normalizes pixel values, and prepares the tensor for the neural network
03
CNN Classification
MobileNetV2 architecture fine-tuned on the DIBaS bacterial image dataset runs inference and outputs class probabilities
04
Results + Context
Species name, confidence score, gram classification, habitat, associated diseases, and treatment options are displayed

about this project
Built for microbiology

microvis is a web application built as part of a microbiology major project, combining Python-based neural networking with real clinical knowledge of bacterial species.

The model uses MobileNetV2 — a lightweight convolutional neural network architecture — fine-tuned on the DIBaS (Digital Image of Bacteria Species) dataset containing 33 bacterial species captured under light microscopy.
